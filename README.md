# никогда вновь не

двое лучших учеников академии исчезают при загадочных обстоятельствах, а спустя неделю Марика возвращается одна. поисками Дракона занимается полиция, но девушка решительно настроена разобраться в произошедшем сама. вот только магии у нее больше нет, а Дракон судя по ее спутанным воспоминаниям-снам — мертв.
